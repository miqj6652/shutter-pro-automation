## Shutter Pro 是什么？

Shutter Pro 是一款跑在 Windows 上的多功能定时任务执行器，由 den4b（Furious Technologies）开发，迭代了整整 20 年，目前最新版本是 v4.8。

它的逻辑特别简单：**设定「触发条件」→ 到了就执行「指定动作」**。

你可以设置的触发条件包括：

- 倒计时（比如 1 小时后）
- 到达某个具体时间点
- CPU/硬盘/网络使用率降到某个阈值
- 电池电量低于某个百分比
- 某个进程结束了
- 某个窗口关闭了
- Ping 断了
- 文件大小达到限制
- ……

触发之后，可以执行的动作也很多，包括：关机、重启、注销、休眠、睡眠、锁屏、屏幕保护、调整音量、播放声音、弹出消息提醒、运行程序、打开文件、关闭窗口、结束进程等，**15+ 种操作**随你挑。

几乎所有「你希望电脑能自己处理的事情」，都可以用 Shutter Pro 来实现。

## 它的几个特别之处

### 多条件逻辑判断

很多定时软件只支持「一个条件触发一个动作」，Shutter Pro 可以同时添加多个触发条件，而且支持三种逻辑：**同时满足全部 / 满足任意一个 / 跟随上一个条件**。

比如你可以设置：「CPU 使用率低于 5%」**且**「距离当前时间已超过 2 小时」，两个条件都满足了才关机。这样就避免了偶尔 CPU 波动一下，结果误触关机的尴尬。

### 预设保存与导出

把常用的任务组合保存为预设，以后一键加载，不用重复设置。Pro 版支持无限预设，还能导出文件分享给其他设备或朋友。

### Web 界面远程管理

这个功能说实话有点超出期待——你可以通过浏览器，**在同一局域网下远程**管理 Shutter Pro：

- 查看电脑当前状态（开机时间、进程列表）
- 实时看到桌面截图
- 远程触发执行预设动作

适合那种家里挂着台式机跑任务、自己在卧室或者公司想随手操控一下的场景。

### 密码保护

给软件设置一个密码，防止任务被别人（或自己手滑）误触启停或篡改。尤其是公用电脑上，这一功能挺实用的。
